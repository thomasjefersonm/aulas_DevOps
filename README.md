# Projeto Frontend

Este projeto consiste no desenvolvimento de uma interface web utilizando React.

O objetivo deste frontend é criar uma aplicação simples que permita a interação do usuário com o sistema, apresentando informações de forma organizada e amigável.

O projeto faz parte da disciplina de DevOps e tem como finalidade demonstrar a utilização de controle de versão com Git, criação de branches e organização do desenvolvimento de software.

Tecnologias utilizadas:

- React
- JavaScript
- HTML
- CSS
- Git e GitHub
